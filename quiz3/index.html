then set the html of the last paragraph to
show it is just text (the red bold is gone).

Test Paragraph.

Answer 

            $("document").ready(function(){
                $("document").ready(function(){
                    $s = $('p').first().detach();
                    $('p').last().append($s.text());
                });
            });

for the level-2 list items

  • II

3. Write jQuery code to select the element that comes immediately before
item three and change its background color to blue.

  • list item 1
  • list item 2
  • list item 3
  • list item 4
  • list item 5

Answer

        $("document").ready(function(){
            $(".third-item").first().prev().css("background-color", "blue");
        });
